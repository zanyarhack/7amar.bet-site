Webhook - پەیوەندی ڕاستەوخۆ بە OKX

ئەم بەشە وا دەکات **کە بەکارهێنەر پارە بنێرێت، ڕاستەوخۆ بۆ ئەکاونتی تۆ بچێت و تۆش زیاد بکەیت بۆ حیسابی ناوەکی.** 
// Webhook endpoint (دەبێت لە سێرڤەری تۆ کار بکات)
POST https://7amar.com/webhook/okx
{
  "user_id": "kurd123",
  "amount": 50.5,
  "currency": "USDT",
  "txid": "0x7a8b3f...",
  "status": "completed"
}

ئەم کۆدەی خوارەوە بکە بە فایلێکی PHP یان Node.js لە سێرڤەرەکەت:

// example Node.js (Express)
app.post('/webhook/okx', (req, res) => {
  const { user_id, amount } = req.body;
  // زیادکردنی پارە بە حیسابی بەکارهێنەر لە داتابەیس
  db.balance.add(user_id, amount);
  res.send('ok');
});

پێشنیاری تایبەت بۆ تۆ

  • API ی OKX: دەتوانیت ڕاستەوخۆ بەستریت بە OKX و بە شێوەی ئۆتۆماتیک دوای هەر تراکنشێک ئاگادار بیت.
  • کلیلێکی تایبەت: بۆ هەر بەکارهێنەر کلیلێکی تایبەت (QR) دروست بکە کە ID ی بەکارهێنەری تێدا بێت. بەمە ئۆتۆماتیک دەستنیشانی دەکات.
  • پشتیوانی تەواوی کوردی: هەموو نووسین و ئاگادارییەکان بە کوردی سۆرانی دەبن.
  • سیستەمی کارتی دیاری: بۆ هەر داخڵکردنێک، دیارییەکی تایبەت (مەسەلەن ١٠٪ بۆنس) زیاد بکە.
  • پەرەپێدانی ئەپلیکەیشنی مۆبایل: بۆ ئاسانکاری زیاتر بۆ بەکارهێنەران.
